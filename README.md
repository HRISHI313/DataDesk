# DataDesk
**Internal Data Operations Platform**

---

## What Is This
DataDesk is an internal tool built for the team to streamline POI data operations. It runs locally on your machine — no internet required for core functionality, no data ever leaves your laptop.

---

## Features
- **Analyser** — upload a POI Excel file and get instant breakdown of construction flags, polygon status, retailer distribution and duplicate ALIs
- **Comparator** — compare two files by ALI or Address, see matched, unmatched and conflicting records
- **Launcher** — distribute POI tasks across teammates based on retailer or record type with balanced workload
